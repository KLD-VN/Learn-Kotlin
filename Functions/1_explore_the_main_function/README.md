# Explore the main() function

```js
fun printHello() {
   println("Hello World")
}

printHello()

// => Hello World
```
Khai báo function -> **fun** keyword + tên của function + **()** chứa arguments (các tham số) + **{}** chứa code bên trong. Không có kiểu tra về cho hàm này vì nó không trả về bất cứ thứ gì.
